# Internal WoPan client

This package is an in-repository copy of `github.com/OpenListTeam/wopan-sdk-go` v0.2.0,
which was already vendored by this project. It is maintained behind the 91 WoPan
driver's internal package boundary so builds and runtime behavior do not depend on
an external OpenListTeam module.
